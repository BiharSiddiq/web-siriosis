Analisis Dataset Penyakit Sirosis Hati

1. Pemahaman Data (Data Understanding)

Dataset ini berisi 418 pasien dengan penyakit sirosis hati dan mencakup 20 kolom, di antaranya:

  • Status: Target klasifikasi (D = Death, C = Censored, CL = Censored Alive)
  • N_Days: Hari sejak diagnosis
  • Drug: Obat yang diberikan (misalnya D-penicillamine atau Placebo)
  • Sex, Ascites, Hepatomegaly, Spiders, Edema: Fitur kategori medis
  • Bilirubin, Albumin, Copper, dll.: Fitur numerik klinis

Beberapa kolom memiliki missing values, terutama: Drug, Ascites, Hepatomegaly, Spiders, Cholesterol, dll.

2. Preprocessing (Pemrosesan Awal)

Kita akan melakukan beberapa langkah pemrosesan awal sebagai berikut:

  • Encoding fitur kategorikal
  • Menyederhanakan target Status menjadi 2 kelas: D (Death) vs Lainnya (hidup/censored)
  • Standarisasi fitur numerik

5. Langkah Pengerjaan Berdasarkan Dataset

Berikut langkah-langkah pengerjaan berdasarkan file cirrhosis.csv:

  1. Membaca file dataset menggunakan pandas.read_csv().
  2. Melihat struktur data dan mengidentifikasi missing values.
  3. Menangani missing values menggunakan imputasi (mean/mode/median).
  4. Menghapus kolom ID yang tidak informatif.
  5. Melakukan encoding pada fitur kategorikal menggunakan LabelEncoder atau OneHotEncoder.
  6. Menyederhanakan kolom Status menjadi biner (1 untuk "D", 0 untuk lainnya).
  7. Melakukan standarisasi fitur numerik menggunakan StandardScaler.
  8. Membagi dataset menjadi data latih (train) dan data uji (test) dengan rasio 80:20.

Langkah-langkah ini dilakukan untuk memastikan data siap digunakan dalam pemodelan machine learning.
